# DSA & LeetCode Solutions

A collection of my solutions to Data Structures and Algorithms problems from LeetCode.

This repository documents my problem-solving journey and serves as a reference for revising important DSA concepts, patterns, and approaches used in coding interviews.

## 📌 Repository Overview

This repository contains solutions covering topics such as:

- Arrays and Strings
- Hashing
- Two Pointers
- Sliding Window
- Binary Search
- Linked Lists
- Stacks and Queues
- Trees and Binary Search Trees
- Heaps and Priority Queues
- Recursion and Backtracking
- Greedy Algorithms
- Dynamic Programming
- Graphs

The repository is continuously updated as I solve and learn new problems.

## 💻 Language

Most solutions are implemented in **Java**, with a focus on:

- Writing clean and readable code
- Understanding the underlying algorithm
- Identifying reusable problem-solving patterns
- Improving time and space complexity
- Building strong DSA fundamentals

## 📂 Repository Structure

Each problem is stored in its own directory with the corresponding solution.

Example structure:

    DSA_LEETCODE/
    ├── 1 Two Sum/
    ├── 11 Container With Most Water/
    ├── 121 Best Time to Buy and Sell Stock/
    ├── 0002-add-two-numbers/
    └── ...



## 🧠 Problem-Solving Approach

For each problem, I try to follow this process:

1. Understand the problem statement and constraints.
2. Develop a brute-force solution when useful.
3. Identify the appropriate data structure or algorithmic pattern.
4. Optimize the solution.
5. Analyze time and space complexity.
6. Revisit important problems for revision.
